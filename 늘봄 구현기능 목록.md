# 늘봄 구현기능 목록

### 1. 사용자 관리 (필수)

### 1.1 회원 인증 및 등급 관리

- [ ]  회원가입 (이메일, 비밀번호) 및 로그인 (JWT 토큰 발급)
- [ ]  **[등급 시스템]** 3단계 접근 권한 구현
    - Level 1 (비회원): 챗봇, 복지 검색 접근 가능
    - Level 2 (일반 회원): 복지 정보 북마크(저장) 기능 사용 가능
    - Level 3 (검증 회원): 서술형 심사 통과 후 커뮤니티 접근 가능

### 1.2 사용자 프로필

- [ ]  내 정보 조회 및 수정
- [ ]  **[맞춤형 정보]** 나이, 거주 지역, 돌봄 대상 정보 입력 (복지 정보 매칭용)

### 2. AI 정서 지원 챗봇 '늘봄' (Core)

### 2.1 실시간 대화 및 상담

- [ ]  LLM 기반 페르소나 챗봇 구현 (공감형 대화)
- [ ]  **[심화]** 긍정 심리학 기반 CBT(인지행동치료) 유도 프롬프트 적용 (감사 일기 등)

### 2.2 위기 감지 시스템

- [ ]  대화 내용 실시간 분석 (Sentiment Analysis)
- [ ]  자살, 학대 등 고위험 키워드/맥락 감지 시 `is_crisis` 플래그 반환
- [ ]  프론트엔드에 보건복지콜센터(129) 연결 팝업 트리거 전송

### 3. AI 맞춤형 복지 정보 매니저 (RAG)

### 3.1 복지 정보 데이터 파이프라인

- [ ]  공공 데이터(복지로, 지자체 사이트) 크롤링 및 수집
- [ ]  **[RAG 구축]** 복잡한 공고문(PDF, HWP) 텍스트 추출 및 Vector DB 임베딩

### 3.2 맞춤형 정보 제공

- [ ]  사용자 프로필(나이, 지역) 기반 필터링 검색
- [ ]  **[쉬운 요약]** LLM을 활용하여 공고문을 17세 수준으로 요약 생성
- [ ]  원클릭 신청 링크 제공
- [ ]  북마크(스크랩) 기능

### 4. 검증된 익명 커뮤니티

### 4.1 가입 심사 시스템

- [ ]  커뮤니티 입장 신청 (서술형 질문 제출)
- [ ]  관리자(또는 AI) 승인 프로세스 (맥락 심사)

### 4.2 게시판 기능

- [ ]  익명 게시글/댓글 작성 (CRUD)
- [ ]  **[AI 모니터링]** 게시글 등록 시 AI 위기 감지 시스템 적용
- [ ]  고위험 게시글 탐지 시 관리자 알림 발송
